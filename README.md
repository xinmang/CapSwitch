# CapSwitch

CapSwitch 是一个 ZTools 插件，用于将 CapsLock 映射为短按/双击/长按三种可配置动作。

## 功能
- 短按映射为任意按键（默认 Shift）
- 双击映射为任意按键（默认 Escape）
- 长按映射为任意按键（默认 Ctrl）
- 可配置长按阈值与双击间隔
- 后台常驻运行，设置窗口独立打开

## 使用
1. 在 ZTools 中安装/加载本插件。
2. 搜索并执行 `capswitch` 打开设置窗口。
3. 根据需要调整短按/双击/长按映射与阈值。
4. 关闭设置窗口后插件仍在后台生效。

## 开发
```bash
pnpm install
```

## 说明
- 依赖 `node-global-key-listener` 进行全局键盘监听。
- 部分系统可能需要辅助功能/管理员权限以拦截按键。
